Mot de Bienvenue


Bienvenue sur un site créé pour la matière d'introduction à la programmation web du cours Techniques de l'informatique profil programmation nouveaux médias. Dans ce cours, nous avons une charge de travail de 16 mois, ce n'est pas seulement rapide, mais c'est un cours intensif très bien. J'apprends beaucoup et je suis très enthousiaste à l'idée de faire un stage et d'entrer dans le domaine que j'aime tant: le développement Web. Par conséquent, comme le professeur nous a demandé de créer un site Web avec HTML et CSS et de montrer 3 sujets qui nous avons dans cette session, comme un moyen de de travail, je voudrais vous présenter un peu l'Institut Grasset et vous expliquer les sujets choisis dans ces quelques pages, j'espère que cela vous plaira.


Les images de gauche sont des liens respectivement vers la page d'accueil de l'institut grasset, vers la page du cours que je fais Techniques de l'informatique, profil programmation nouveaux Médias, et vers le stage page de notre collège, cliquez et parcourez ces pages.
